# Status Block Template

Use this block at the end of every major iteration.

## Current stage

- [ ] Research
- [ ] Opportunity selection
- [ ] Validation
- [ ] Requirements/design
- [ ] Planning
- [ ] Implementation
- [ ] Verification/release
- [ ] Retrospective

## Iteration objective

State the narrow goal of this iteration.

## Hypothesis

State what you currently believe and why.

## What would change your mind

State the evidence or result that would cause you to revise the current direction.

## Evidence gathered

List the strongest evidence gathered this iteration.

## Commercial and operability notes

List the most important updates about monetization, support burden, or agent-operability.

## Decisions made

List major decisions made and why.

## Files changed

List files created or updated.

## Tests or checks run

List verification steps completed.

## Risks and unknowns

List the most important unresolved items.

## Gate status

State whether the work is ready for the next human gate or not.

## Next action

State the smallest sensible next step.
