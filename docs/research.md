# Research

Use this file to capture problem-space research.

## Goal

Identify repeated pain, unmet needs, and gaps in current tools within the theme area.

## Suggested sections

- target user segments
- repeated complaints or workflow friction
- evidence snippets and sources
- existing alternatives
- why alternatives are insufficient
- opportunity candidates
- open questions

## Quality bar

The file should make it obvious that the opportunity ideas came from real observations rather than pure imagination.
