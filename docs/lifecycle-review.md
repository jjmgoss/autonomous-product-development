# Lifecycle Review

Use this file at the end of the run to assess lifecycle completeness.

## Questions

- Did research actually shape the product choice?
- Was validation real or performative?
- Did requirements trace cleanly into implementation?
- Was verification good enough to trust the claims made?
- What lifecycle areas remain weak?
- What would need to improve before a real product push?

## Output

End with one recommendation:

- iterate
- pivot
- pause
- archive
