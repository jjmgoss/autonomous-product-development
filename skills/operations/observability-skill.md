# Observability Skill

## Goal
Ensure the MVP is observable and issues can be detected quickly.

## When to Use
During implementation and before deployment.

## Procedure
1. Add basic logging and monitoring.
2. Define key metrics to track.
3. Document observability setup in `docs/operations-gap-analysis.md`.

## Expected Outputs
- Observability plan
- Metrics list

## Common Failure Modes
- No monitoring or logging
- Missing key metrics
