# Incident Readiness Skill

## Goal
Prepare for and respond to incidents during MVP operation.

## When to Use
Before and after deployment.

## Procedure
1. Define incident response plan.
2. Document escalation and rollback steps.
3. Review readiness in `docs/operations-gap-analysis.md`.

## Expected Outputs
- Incident response plan
- Rollback steps

## Common Failure Modes
- No incident plan
- No rollback procedure
