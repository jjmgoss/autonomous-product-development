# Feedback Loop Skill

## Goal
Collect and act on user and system feedback.

## When to Use
After MVP is in use or tested.

## Procedure
1. Define feedback channels (user, system, metrics).
2. Collect and review feedback regularly.
3. Document findings in `docs/metrics-and-feedback.md`.

## Expected Outputs
- Feedback summary
- Action items

## Common Failure Modes
- Ignoring feedback
- Not closing the loop on issues
