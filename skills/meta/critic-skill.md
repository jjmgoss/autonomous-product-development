# Critic Skill

## Goal
Detect weak reasoning, overscoping, and stalled progress.

## When to Use
After each major stage or when stuck.

## Procedure
1. Review recent decisions and evidence.
2. Identify weak logic, missing evidence, or scope creep.
3. Recommend corrections or next steps.

## Expected Outputs
- Critique summary
- Action recommendations

## Common Failure Modes
- Ignoring weak evidence
- Failing to recommend course corrections
