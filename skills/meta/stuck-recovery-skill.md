# Stuck Recovery Skill

## Goal
Help the agent recover when progress stalls.

## When to Use
When blocked, confused, or making no progress.

## Procedure
1. Diagnose the cause of being stuck.
2. Brainstorm possible next steps or pivots.
3. Select the smallest experiment to regain momentum.

## Expected Outputs
- Diagnosis summary
- Next-step experiment

## Common Failure Modes
- Spinning in circles
- Not seeking help or changing approach
